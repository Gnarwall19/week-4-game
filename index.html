Crystal Collector!

You will be given a random number at the start of the game.

There are four crystals below. When a crystal is clicked,
a specific number of points will be added to your total score.

The game is one when your Current Number matches
the Target Number exactly. The game is lost if your
Current Number exceeds the Target Number.

The number value of each crystal will be hidden from you
and can only be made known by clicking it and deducing it's
value based on the amount added to your Current Number.

Each crystal will be assigned a new number at random by
the start of each round.

Good Luck.
